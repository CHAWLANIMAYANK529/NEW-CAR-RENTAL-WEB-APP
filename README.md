# 🚗 Car Rental Website

A modern, responsive, and user-friendly Car Rental Website built using HTML, CSS, and JavaScript. This project showcases a complete front-end solution for a car rental business with booking integration and premium UI design.

---

## 🔥 Features

- 🚘 Car Fleet Listing with detailed cards
- 📱 WhatsApp Booking Integration (Auto-filled messages)
- 📞 Direct Call Option
- ⭐ Customer Reviews Section with auto-scroll animation
- 🎯 "Why Choose Us" premium section
- 📩 Contact Form Integration
- 📜 Privacy Policy, Terms of Use & Refund Policy pages
- 💻 Fully Responsive Design
- 🎨 Modern UI with animations and hover effects

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap (optional)
- Font Awesome Icons
